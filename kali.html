كيفية تنزيل Kali Linux

شرح عملي لتثبيت نظام Kali Linux على الحاسوب مع الأدوات المهمة للمطورين، وأفضل الممارسات للتثبيت الآمن.

تثبيت Kali Linux على VMware خطوة بخطوة

  1. تحميل نسخة Kali Linux الرسمية (ملف الـ ISO):
    افتح موقع Kali Linux الرسمي: https://www.kali.org/get-kali/
    اختَر النسخة المناسبة (64-بت غالبًا) وحمّل ملف الـ ISO.
  2. تحميل وتثبيت برنامج VMware Workstation Player:
    اذهب لموقع VMware لتحميل النسخة المجانية: https://www.vmware.com
    حمّل البرنامج المناسب لنظام تشغيلك (ويندوز أو لينكس) وثبّته.
  3. إنشاء جهاز افتراضي جديد (New Virtual Machine):
    افتح VMware Workstation Player، اضغط على "Create a New Virtual Machine"، ثم اختَر ملف الـ ISO الخاص بـ Kali Linux.
    تابع الخطوات واضغط Next.
  4. إعدادات الجهاز الافتراضي:
    - سمِّ الجهاز الافتراضي (مثلاً "Kali Linux").
    - اختَر مكان التخزين.
    - حدد حجم القرص (20 جيجابايت أو أكثر).
    - اختر "Split virtual disk into multiple files".
    - اضغط Next ثم Finish.
  5. ضبط موارد الجهاز (RAM والمعالج):
    افتح إعدادات الجهاز الافتراضي وزد الرام لـ 2 جيجابايت (2048 ميجابايت) أو أكثر.
    يمكنك زيادة عدد أنوية المعالج إذا أردت.
  6. تشغيل الجهاز الافتراضي وتثبيت Kali Linux:
    شغّل الجهاز الافتراضي.
    اختر "Graphical install" وابدأ التثبيت باتباع التعليمات (اختيار اللغة، المنطقة الزمنية، تقسيم القرص، إنشاء المستخدم).
  7. إكمال التثبيت وإعادة التشغيل:
    بعد الانتهاء، أعد تشغيل الجهاز الافتراضي.
    ستبدأ Kali Linux بالعمل داخل VMware.
  8. استخدام Kali Linux بأمان داخل VMware:
    يمكنك تجربة الأدوات والبرامج بأمان دون التأثير على نظام تشغيلك الأساسي.

نصائح مهمة:

  • حافظ على تحديث Kali Linux باستمرار.
  • استخدم الجهاز الافتراضي فقط للأغراض القانونية.
  • قم بتثبيت أدوات VMware Tools لتحسين الأداء.
